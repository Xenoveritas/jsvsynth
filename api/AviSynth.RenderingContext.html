Class: RenderingContext

AviSynth. RenderingContext

new RenderingContext()

A rendering context capable of writing on a frame. This class cannot be constructed directly, instead use AviSynth.VideoFrame#getContext. Currently the only supported RenderingContext is AviSynth.SimpleRenderingContext.
