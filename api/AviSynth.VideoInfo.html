Class: VideoInfo

AviSynth. VideoInfo

new VideoInfo()

This functions as the base class for both AviSynth.Clip and AviSynth.Filter. It provides access to the video information for a clip. (Clips and filters are effectively the same thing.)

Most of the field names are based on the clip property getters that AviSynth provides.
